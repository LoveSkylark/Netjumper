
# LibreNMS-Netjumper
Templet for an Ubuntu server to run as Jump box for a cutomer.
- SSH & TELNET access to Monitored devices.
- Basic network toolbox (mtr, trace, ping throughput calc)
- Scripts to gather infromation (device list, serial numbers, unknown XDP neighbours)

### Install
- contains a list of requiremnets for the script to run

### script 
- script that you can run with Librenms

### profile.d 
if .sh file is copied into /etc/profile.d/ folder then SSH and other command will autocomplete using the known host extracted from the script.
